Проект собирается командой "mvn clean package" из корневой директории. Сборка находится в папке target. 
Для запуска приложения нужно из директории target выполнить команду java -jar exchange-1.0-SNAPSHOT.jar имя_файла_с_клиентами имя_файла_с_заявками имя_файла_для_вывода_результатов.
Для работы программе нужны сторонние библиотеки (log4j), они находятся в папке lib в директории target, classpath прописан в манифесте (как и main class).
В процессе работы программа создает директорию log с файлом log.log, куда выводит некоторую отладочную информацию.
